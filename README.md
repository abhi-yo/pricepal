# Serene Landscapes Explorer

A modern web experience showcasing beautiful natural landscapes with immersive visual effects.

## Features

- Dynamic particle background system
- Responsive hero section with texture overlays
- Sophisticated typography using Geist fonts
- Smooth theme transitions (light/dark mode support)
- Optimized performance with Next.js

## Installation

```bash
npm install
# or
pnpm install
```

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Next Themes (Dark mode handling)
- Canvas API (Particle effects)

## Running the Application

```bash
npm run dev
# or
pnpm dev
```
